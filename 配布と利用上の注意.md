# 「音感クルーズ」配布・利用に関する注意事項

この文書は、**アプリを配布・公開する側（あなた）**と、**アプリを使う側（利用者）**の双方が、リスクや前提を共有できるようにまとめたものです。法的な助言ではありません。必要に応じて専門家（弁護士・税理士など）に相談してください。

---

## 1. まず共通で知っておくこと

### 1.1 このアプリの性質

- **ブラウザ上で動く Web アプリ（PWA 対応）**です。インストール型のネイティブアプリ（App Store / Google Play 経由の正式アプリ）とは仕組みが異なります。
- **主な用途**: アコースティックギター等の練習向けの、音感トレーニングの補助です。
- **医療・聴力検査・公式の音楽教育の代替**ではありません。

### 1.2 データの扱い（技術的な前提）

- 設定や Pro 版のカスタム内容などは、**利用者の端末の中（ブラウザの保存領域）**に保存される設計です。**あなたのサーバーに利用者の名前やメールを送る仕組みは、このプログラムだけではありません**（※別途アクセス解析タグ等を埋め込んでいない限り）。
- その代わり、**端末の故障・ブラウザのデータ削除・アプリ削除**で、保存内容が消えることがあります。バックアップは利用者側の環境に依存します。

### 1.3 動作環境・音の精度

- 音の再生は **Web Audio** 等に依存します。**端末・ブラウザ・ヘッドフォン**によって、音程やタイミングの体感が変わることがあります。
- 「絶対に正しい基準音」として機器に認定する用途には向きません。

## 3. 利用者（アプリを使う人）向け

### 3.1 利用目的

- 本アプリは、**楽器練習の補助**を目的としています。**医療機器・聴力の医学的診断・公式試験対策の保証**はありません。

### 3.2 保証の限界

- **バグ・表示崩れ・音が鳴らない**などが起きても、**損害の補償や結果の保証はできません**（配布者が別に規約で定める場合を除く）。
- 利用は**自己責任**で、端末の設定や音量に注意してください。長時間の大音量は聴力に影響することがあります。

### 3.3 データの保存について

- 設定やカスタム内容は**端末内に保存**される想定です。**機種変更・ブラウザのデータ削除・キャッシュ削除**で消えることがあります。重要な内容は別途メモするなど、利用者側のバックアップを推奨します。

### 3.4 動作環境

- **推奨**: 比較的新しい **Chrome（Android）** や **Safari（iPhone）** など。古い端末・ブラウザでは動作が不安定なことがあります。
- **インターネット接続**: 初回の読み込みや更新時に必要です。オフライン動作は環境やキャッシュの状態に依存します。

### 3.5 Pro 版・ベータ版を案内された場合

- **共有された URL** からアクセスしてください。URL を第三者に再共有するかどうかは、配布者のルールに従ってください。
- ベータ版は**試用・テスト**向けで、本番版と動きや範囲が違うことがあります。

### 3.6 禁止・注意が望ましい行為（例）

- 配布者や第三者の権利を侵害する利用。
- サーバーへの過度な負荷をかける自動アクセス（通常の個人利用の範囲を超える場合）。
- 違法な目的での利用。

※ 詳細な禁止事項を定める場合は、配布者が別途「利用規約」を用意してください。
