# 音感トレーニング (Pitch Trainer)

## プロジェクト概要
Webブラウザで動作する絶対音感トレーニングアプリです。
メロディ（単音）とコード（和音）の聞き取り練習ができます。

## 実装状況 (Current Status)

### 最新の修正 (2026-02-19)
- **レイアウト修正**: 画面が左右に分割される不具合を修正し、スマートフォンライクなシングルカラムレイアウトに統一しました。
- **音声バグ修正**: ゲーム開始時(`startGame`)の初期化処理を修復し、音が正しく出るようにしました。
- **コンボ表示**: スコア・ストリーク表示を廃止し、連続正解数を示す「COMBO」のみのシンプル表示に変更しました。

### 機能
- **メロディ音感**: Stage 1〜6 (ドミソ等、音数を徐々に増やして出題)
- **コード音感**: Stage 7〜 (C, F, G, Dm 等のコード聞き取り)
- **設定**: 
  - 再生速度変更
  - 余韻（サスティン）調整
  - オクターブ変更
  - 楽器変更 (Piano, Violin, Guitar)

## 技術スタック
- HTML5
- CSS3 (Vanilla CSS, variables used for theming)
- JavaScript (Vanilla JS, Web Audio API)

## 起動方法
ローカルサーバーを立てて `index.html` を開いてください。

```bash
# プロジェクトルートで実行
python3 -m http.server 8080
```

ブラウザで `http://localhost:8080` にアクセスします。
(キャッシュが残っている場合はクエリパラメータをつけてください: `?v=final23` など)
