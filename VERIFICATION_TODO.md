# 動作確認チェックリスト (ユーザー用 Todo)

後日、以下の手順で正しく動作しているか確認してください。

## 1. 準備：サーバーの起動
Macでアプリを動かすための「サーバー」を起動する必要があります。
1. Macのターミナル（またはこのチャット画面）で以下のコマンドを実行します。
   ```bash
   python3 -m http.server 8081
   ```
   ※ すでに「Running terminal commands」として表示されている場合は、そのままで大丈夫です。

---

## 2. パソコンでの確認（PC）
- [ ] **設定の保存テスト**
  1. [http://localhost:8081](http://localhost:8081) を開く。
  2. 歯車アイコンから設定（楽器をピアノ、スピードを1.5xなど）を変更する。
  3. ブラウザを更新（再読み込み）する。
  4. 設定がそのまま残っていることを確認する。
- [ ] **エラーの確認**
  1. 画面が固まったりせず、スムーズに起動することを確認する。

---

## 3. スマホでの確認（Mobile）
※ Macとスマホが同じWi-Fiに繋がっている必要があります。

- [ ] **スマホでアクセス**
  1. スマホのブラウザで `http://192.168.68.112:8081` を開く。
- [ ] **PWA（アプリ化）の確認**
  1. iPhoneは「ホーム画面に追加」、Androidは「アプリをインストール」を実行。
  2. ホーム画面に専用のアイコンができることを確認。
  3. ホーム画面のアイコンから起動した際、全画面でアプリのように動くことを確認。
- [ ] **操作感の確認**
  1. 音ボタンをタップしたとき、視覚的なフィードバック（ボタンが沈む動き）があるか確認。
  2. 音が正しく出るか確認（iPhoneの場合はマナーモードを確認）。

---

## 4. PWAアイコンの確認
- [ ] **アイコンのデザイン**
  1. ホーム画面に追加されたアイコンが、新しく作ったモダンなデザインになっているか確認。
