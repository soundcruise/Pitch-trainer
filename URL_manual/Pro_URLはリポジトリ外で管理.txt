■ Pro版の公開URL（会員向け）の置き場所について

GitHub の README やこのフォルダからは、**Pro のフル URL（https://… から始まるアドレス）を削除**しました。
会員にだけ伝える URL は、次のような **リポジトリに載せない場所** にメモしてください。

例:
・スマホ・PC のメモアプリ（同期のみ自分用）
・YouTube メンバー限定の投稿・コミュニティ（本文に URL を書く）
・会員向けの Google ドキュメント（共有範囲を限定）

メモの例（中身は自分だけが書く）:
--------------------------------------------------
  音感クルーズ Pro（GitHub Pages）
  https://（ここに実際のフル URL をあなただけが記入）
--------------------------------------------------

※ 本リポジトリのソースコード上には、Pro 用のフォルダ名などが残ります。
  「完全に誰にも分からない」というより、README にフル URL を載せない運用です。
